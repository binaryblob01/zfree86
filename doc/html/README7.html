7. Reporting Bugs

Bugs should be reported to XFree86@XFree86.org . Before reporting bugs, please check the XFree86 server log file, which can be found at /var/log/XFree86.0.log on most platforms. If you can't resolve the problem yourself, send the entire log file with your bug report but not the operating system core dump. Do not edit the log file as our developers use it to reproduce and debug your problem.
