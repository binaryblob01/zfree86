1. What is XFree86

XFree86 is a port of X11R6.5.1 that supports several versions of Unix. It is derived from X386 1.2 which was the X server distributed with X11R5. This release consists of many new features and performance improvements as well as many bug fixes. The release is available as a source code distribution, as well as binary distributions for many architectures.

The sources for XFree86 are available by anonymous ftp from:

ftp://ftp.XFree86.org/pub/XFree86/current
Solaris binaries for XFree86 are available for anonymous ftp at the same address. Currently, two binary distributions are available: one for Solaris 8 x86, the other for previous Solaris x86 releases. They are not interchangeable. No binary distribution is available for Solaris/SPARC as it is still under development.
