2. Introduction

This document describes how to download, compile and install the DRI. The DRI provides 3D graphics hardware acceleration for the XFree86 project. This information is intended for experienced Linux developers. Beginners are probably better off installing precompiled packages.

Edits, corrections and updates to this document may be mailed to brian@tungstengraphics.com .
