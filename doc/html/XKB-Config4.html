4. Keymap XKB Configuration

It is the formerly used way to configure xkb. The user included a special keymap file which specified the direct xkb configuration. This method has been obsoleted by previously described rules files which are far more flexible and allow simpler and more intuitive syntax. It is preserved merely for compatibility reasons. Avoid using it if it is possible.
