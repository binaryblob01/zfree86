README for XFree86® 4.8.0

The XFree86 Project, Inc

August 2007

XFree86(R) is an Open Source version of the X Window System that supports many UNIX(R) and UNIX-like operating systems (such as Linux, FreeBSD, NetBSD, OpenBSD and Solaris) on Intel and other platforms. This version is compatible with X11R6.6.
