1. Support chipsets

The apm driver in the SVGA server is for Alliance Promotion graphics chipsets. The following chipsets are supported:

  • 6422 Old chipset. The driver is still very unstable and has computer crashes. You should use XFree86 3.3.x for this chipset.
  • AT24 As found in Diamond Stealth Video 2500. Quite similar to AT3D.
  • AT25, AT3D AT3D is found in Hercules Stingray 128/3D. Most other Voodoo Rush based cards use the AT25 which is identical except it doesn't have the 3D stuff in it.
