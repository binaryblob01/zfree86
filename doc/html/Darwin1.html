1. Introduction

XFree86, a freely redistributable open-source implementation of the X Window System, has been ported to Darwin and Mac OS X. This document is a collection of information for anyone running XFree86 on Apple's next generation operating system.

Most of the current work on XFree86 for Darwin and Mac OS X is centered around the XonX project at SourceForge. If you are interested in up-to-date status, want to report a bug, or are interested in working on XFree86 for Darwin, stop by the XonX project.
